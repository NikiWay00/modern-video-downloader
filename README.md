Funzionalita principali

- Download video MP4 (con audio compatibile AAC)
- Download audio MP3
- Coda download multipla
- Barra di avanzamento dettagliata
- Compatibilita con Windows Media Player e VLC
- FFmpeg incluso (nessuna installazione esterna)
- Interfaccia moderna (Dark Mode)
- Versione portable (nessuna installazione)
- EXE standalone

---------------------------------------------------------------------

Requisiti di sistema

- Sistema operativo: Windows 10 / Windows 11
- Architettura: 64-bit
- Connessione Internet: necessaria per il download dei contenuti

Non e necessario installare Python, FFmpeg o altre dipendenze.

--------------------------------------------------------------------

Come usare il programma (EXE)

- Avvia ModernVideoDownloader.exe
- Incolla l'URL del video nel campo apposito
- Seleziona Video (MP4) oppure Audio (MP3)
- Scegli la cartella di destinazione
- Premi Avvia Download
- Attendi il completamento
- Il file finale verra salvato nella cartella scelta

--------------------------------------------------------------------

Come eseguire da sorgente (sviluppo)

1) Crea un virtualenv e installa le dipendenze
   - pip install -r requirements.txt
2) Avvia l'app
   - python run.py

-------------------------------------------------------------------

Note importanti

- Durante il download potresti vedere file temporanei (audio + video separati): e normale.
- Vengono uniti automaticamente a fine download.
- Il programma non aggira DRM o protezioni.
- Usa solo contenuti che hai il diritto di scaricare.
